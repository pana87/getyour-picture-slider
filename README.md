# Simple Picture Slider

## Install

```bash
npm i github:pnts87/getyour-picture-slider
```

## Usage

```html
  <getyour-picture-slider>
    <img id="some-picture" src="https://picsum.photos/id/1/500" />
    <img id="some-picture" src="https://picsum.photos/id/2/500" />
    <img id="some-picture" src="https://picsum.photos/id/3/500" />
    <img id="some-picture" src="https://picsum.photos/id/4/500" />
    <img id="some-picture" src="https://picsum.photos/id/5/500" />
    <img id="some-picture" src="https://picsum.photos/id/6/500" />
  </getyour-picture-slider>

  <script type="module" src="./getyour-picture-slider.js"></script>
```
